Privacy Policy for Motivation Lo Bhai

1. Data Collection

This extension does not collect, store, or transmit any personal information including:

  • Browser history or activity
  • Personal identifiers
  • Location data
  • Any other sensitive information

2. External Services

The extension fetches motivational quotes from these third-party APIs:

  • ZenQuotes API
  • Quotable API

These requests are anonymous and contain no user-identifiable information.

3. Permissions

The extension requires:

  • host_permissions: Only to access the quote APIs listed above

No other permissions are requested or required.

4. Changes to This Policy

Any updates to this policy will be posted here with a revised "Last Updated" date.
